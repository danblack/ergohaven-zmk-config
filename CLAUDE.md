# CLAUDE.md

Гайд для Claude Code по этому репозиторию. Кратко — детали для пользователя в `README.md`.

## Что это

Личный ZMK-конфиг для клавиатур Ergohaven (board `ergohaven`). Сборка — только
через GitHub Actions (`ergohaven/ergohaven-zmk` reusable workflow). **Локального
ZMK-тулчейна нет** — не пытайся компилировать локально; единственная проверка
правок = успешная сборка в Actions после пуша. Цели сборки перечислены в
`build.yaml` (закомментированные блоки = выключенные цели). Сейчас активна только
**op36**.

## Структура

- `config/<board>.keymap` — devicetree: `behaviors`, `combos`, `macros`, `keymap`.
- `config/<board>.conf` — Kconfig прошивки.
- `config/<board>.json` — физический layout (порядок ключей; для op36 — 36 позиций).
- `config/keys_ru.h` — макросы `RU_*` для кириллицы.
- `*_ruen.keymap`, `*_qube.conf` — варианты раскладок/донгла; часть `.json` —
  симлинки на базовый.

## Конвенции

- Behaviors объявляются в `/ { behaviors { ... } }`; держи выравнивание столбцов
  в массивах `bindings` keymap'а — оно ручное и значимо для читаемости diff'ов.
- Hold-tap'ы именуются по схеме «что-hold + что-tap»: `mosk` (`&mo`/`&sk`),
  `mosl` (`&mo`/`&sl`), `ml` (`&mo`/`&kp`), `tomo` (`&mo`/`&to`). Прежде чем
  заводить новый behavior — проверь, нет ли подходящего среди существующих
  (часто нет: пары bindings отличаются именно tap-веткой).
- Номера слоёв жёстко зашиты в биндингах (`&mo 13`, `&to 0` и т.п.) — при
  добавлении/перестановке слоя проверь все ссылки. Полный список слоёв op36 — в
  `README.md`.

## op36: важные нюансы

- **Нет указательного устройства.** Мышь/скролл эмулируются клавишами `&mmv`,
  `&msc`, `&mkp` на слоях `MUS` (13) и `SMUS` (16). Поэтому input-processor'ы
  (`zip_*`, `temp-layer`) из velvet-конфигов здесь неприменимы — нет физического
  ввода, на который они реагируют.
- Чтобы отправить событие мыши, нужный слой должен быть активен. «Быстрый скролл
  без полного переключения» = momentary-слой (`&mo`).
- Вход на мышиный слой с NAV: `&tomo 13 13` — tap даёт полное `&to 13`, hold —
  momentary `&mo 13`.
